Langkah Analisis Grup Konten WhatsApp

  1. Data dari WhatsApp dieksport dan digabungkan menjadi sebuah file teks (contoh: data_group.txt).
  2. File teks tadi diubah menggunakan aplikasi Excel dan disimpan menjadi file csv (contoh: data_group.csv).
  3. File CSV dikompres menjadi TAR (contoh: data_group.tar).
  4. Data dibersihkan dengan hanya mengambil angka, huruf, dan tanda baca tulis umum.
  5. Menggunakan Visual Studio Code, data diclustering menggunakan Metaflow. Algoritma clustering yang digunakan untuk mengelompokkan data menjadi 3 yaitu kMeans.
  6. Kata-kata teratas dalam setiap cluster dianalisis untuk mengidentifikasi tema.

Hasil Clustering

3 Clusters

    Kata-kata yang dihasilkan:
    ["eko", "24", "ums"]

4 Clusters

    Kata-kata yang dihasilkan:
    ["media", "omitted", "bi", "eko"]

5 Clusters

    Kata-kata yang dihasilkan:
    ["invite", "joined", "using", "group", "link"]

Kesimpulan

Dari hasil clustering tersebut, dapat diambil kesimpulan sebagai berikut :
- Clustering 3 berisikan tentang informasi dosen yang mengajar di kampus tersebut
- Clustering 4 berisikan tentang materi yang diajarkan oleh pak dosen Eko (mata kuliah Bahasa Indonesia)
- Clustering 5 berisikan tentang link ajakan untuk masuk ke grup mata kuliah
